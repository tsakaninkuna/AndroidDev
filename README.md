#TipApp

The project is all about a tip app which is created using android studio. The TippApp has a base, a percentage with indication to show case how much tip you can offer to the waiter at any resturant. The features are poor, good, average and amazing. The elements used are Edittext, TextView, SeekBar and a button.
Here you will enjoy to see how muhc generous can you tip a worker at a resturant up to 50% tip. Do you offer tips?
